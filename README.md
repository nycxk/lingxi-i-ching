# 你想预知的下一个结果，何必求神仙-灵犀六爻SKILL

一个把传统六爻流程工程化的命令行小工具。

它会认真摇卦、老实排盘、尽量讲人话，不负责替你替天行道。

## 一句话介绍

`lingxi-i-ching.SKILL` 是一个带点玄学气质、但尽量保持工程理性的六爻 CLI：

- 三钱起卦，支持随机种子复现
- 自动生成本卦、变卦、六亲、六神、世应、旬空
- 根据用神与生克关系输出规则化断语
- 可选接入 LLM，把术语翻译成更像人话的建议
- 自动保存历史记录，方便回看“当时为什么会这么想”

## 核心能力

1. 用三枚铜钱模拟六次投掷，自下而上生成六爻。
2. 自动计算本卦、变卦、六亲、六神、世应和旬空。
3. 根据问题关键词自动推定用神，例如事业看 `官鬼`，财运看 `妻财`。
4. 用简化规则引擎给出评分、断语和行动建议。
5. 可选调用 OpenAI 接口，把“术语味”稍微降一点。

## 快速开始

### 直接运行

```bash
python scripts/liuyao.py --question "我该不该换工作"
```

### 指定随机种子，方便复盘

```bash
python scripts/liuyao.py --question "这次合作能顺利推进吗" --seed 7
```

### 启用 LLM 建议

```bash
python scripts/liuyao.py --question "近期财运如何" --llm --model gpt-4o-mini
```

## 参数说明

| 参数 | 说明 |
| --- | --- |
| `--question` | 占问主题，建议填写得越具体越好 |
| `--seed` | 随机种子，用于复现实验 |
| `--llm` | 是否启用 LLM 增强建议 |
| `--model` | LLM 模型名，默认 `gpt-4o-mini` |
| `--history` | 历史记录输出路径 |
| `--gua-db` | 卦库 JSON 路径，未提供时会退回卦码名 |

## 示例输出风格
